LIÇÃO 18 — Quantities and Choices

Nível A1
🎯 Objetivo da Lição

• Usar Some em frases afirmativas e ofertas educadas, e Any em negativas e perguntas.

• Aprender determinadores: every, each, all, both, either, neither.

• Ampliar vocabulário de família, escola, compras e pessoas.

🔥 Warm-Up: Some ou Any?

Complete mentalmente:

1. I have ___ apples. (afirmativo) → some

2. I don't have ___ bananas. (negativo) → any

3. Do you have ___ milk? (pergunta) → any

4. Would you like ___ coffee? (oferta) → some

5. There aren't ___ eggs. (negativo) → any

💡 Dica: "Some" = algum/alguns. "Any" = algum/nenhum (em negativas e perguntas). Mas ofertas usam "some"!

🗣️ Diálogo: Two Friends Shopping Together

Anna: Let's go shopping! I need some new clothes.

Beth: OK! Do you need any shoes?

Anna: No, I don't need any shoes. But I want some T-shirts.

Beth: Would you like some help finding them?

Anna: Yes, please! Look — both of these shirts are nice!

Beth: They are! Which one do you prefer? Either the blue or the red?

Anna: I can't decide! Neither is cheap enough. How much are they?

Beth: All the T-shirts are 30% off today!

Anna: Great! I'll take both! Do you have any discount cards?

Beth: No, I don't. But every customer gets a receipt.

🔁 Repita com atenção: Observe some/any, both/either/neither, all/every!

📘 Gramática em Ação: Some / Any & Determiners
1. Some vs. Any
SomeAny
Afirmativo✅ I have some apples.❌
Negativo❌✅ I don't have any apples.
Pergunta❌ (geralmente)✅ Do you have any apples?
Oferta educada✅ Would you like some coffee?❌

I need some milk. | She doesn't have any money. | Do you have any questions? | Would you like some tea?

2. Determinadores

A. every (cada/todo): Every student has a book. (cada estudante tem um livro)

B. each (cada — individual): Each person gets a prize. (cada pessoa ganha um prêmio)

C. all (todos): All the students passed. (todos os estudantes passaram)

D. both (ambos): Both options are good. (ambas opções são boas)

E. either (qualquer um dos dois): You can take either bus. (você pode pegar qualquer um dos dois ônibus)

F. neither (nenhum dos dois): Neither answer is correct. (nenhuma das duas respostas está correta)

Dica: Either/Neither são para 2 opções. All/None são para 3+ opções.

D. Verdadeiro ou Falso?
"Would you like any tea?" — Oferta educada correta.
"Both" é usado para 2 opções.
"Neither" significa "qualquer um dos dois".
"I don't have any money" — Frase correta.
🎭 Prática Oral / Role-Play

A: I need some new clothes. Do you have any suggestions?

B: Sure! Both of these shirts look great on you.

A: Thanks! But I don't like either color. Do you have any other options?

B: Let me check. Would you like some help?

A: Yes, please! All the clothes here are nice, but I need some that are cheaper.

B: Every item on this rack is 20% off!

A: Neither of the first two was my size. Can I try this one?

B: Of course! Each fitting room is available.

👅 Trava-língua

Neither nurse nor neighbor needs new knives.

🔁 Repita 3x rápido! Concentre-se no som "n".

🌍 Curiosidade Cultural: Shopping in the US

Nos EUA, o preço na etiqueta NÃO inclui imposto! O sales tax (imposto de vendas) é adicionado no caixa — geralmente 5-10% a mais. Isso surpreende muitos estrangeiros!

Outra diferença: nos EUA, gorgetas (tips) são quase obrigatórias em restaurantes — 15-20% da conta. Não deixar gorjeta é muito indelicado!

💡 Dica: "Keep the change" significa "fique com o troco" — é comum quando o troco é pequeno.
